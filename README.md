# insurance-risk-analysis

Analyze car insurance data to find low-risk customers and optimize pricing.

## Files

- `notebooks/01_eda.ipynb` - Data exploration
- `notebooks/02_hypothesis_testing.ipynb` - Statistical tests
- `notebooks/03_modeling.ipynb` - Prediction models
- `src/` - Python code files
- `data/` - Insurance dataset

## Run

```bash
pip install -r requirements.txt
jupyter notebook
```
